# 🧾 Purchasing Control System

> 個人の購入履歴・支出管理を目的としたWebアプリケーション

---

## 📌 概要
本プロジェクトは、日常の購入履歴や支出情報を管理するために開発した  
**購入管理Webアプリケーション**です。

ERD設計をベースに、データの構造化と効率的な管理を重視して開発しました。

---

## 🚀 主な機能
- 🧾 購入履歴の登録・管理  
- 📊 支出データの一覧表示  
- 🔍 条件検索（期間・カテゴリ別）  
- 🗂 カテゴリ別管理機能  
- 📁 データの保存・履歴管理  

---

## 🛠 Tech Stack
**Backend**
- Java  
- Spring Framework  

**Frontend**
- HTML / CSS / JavaScript  
- JSP  

**Database**
- Oracle  

---

## 💡 特徴
- 🧩 ERD設計をベースとしたデータ構造設計  
- 📊 データ管理に最適化されたDB設計  
- 🔎 検索・フィルタリング機能による利便性向上  
- 💻 実務を意識したCRUD機能実装  

---

## 🎯 工夫した点
- データの一貫性を保つために  
  **正規化を意識したERD設計を実施**
- 実際の利用を想定し、  
  **検索・分類機能を重視して設計**
- 将来的な拡張を考慮し、  
  **柔軟なDB構造を構築**
