# Cursor 사용 매뉴얼 (초보자용)

이 파일은 집에서 Cursor를 사용할 때 옆에 띄워두고 그대로 따라가기 위한 파일입니다.

[기본 원칙]
1. 한 번에 전체 앱을 만들라고 하지 않는다.
2. 문서를 먼저 읽게 한다.
3. 08_CURSOR_MASTER_PROMPT.md로 시작한다.
4. 09_PHASE_GUIDE_AND_ACCEPTANCE.md 순서대로 진행한다.
5. 각 Phase 결과를 꼭 확인하고 다음으로 넘어간다.

[가장 추천하는 작업 순서]
1. handoff zip 압축을 푼다.
2. 00_START_HERE_READ_FIRST.md를 읽는다.
3. Cursor에서 새 프로젝트 폴더를 연다.
4. Cursor에 아래처럼 입력한다:
   이 폴더 안 문서를 전부 읽고 앱 구조와 기능을 한국어로 요약해줘. 코드는 아직 만들지 마.
5. 요약이 맞으면 08_CURSOR_MASTER_PROMPT.md 내용을 Cursor Agent에 넣는다.
6. 프로젝트 뼈대를 만든 뒤 09_PHASE_GUIDE_AND_ACCEPTANCE.md를 보고 Phase 1부터 순서대로 진행한다.

[항상 기억할 것]
- 결과 확인 없이 다음 단계로 넘어가지 않는다.
- Cursor가 만든 구조가 문서와 다르면 바로 수정 요청한다.
- 복잡한 요구는 작은 작업으로 쪼개서 지시한다.

[좋은 수정 요청 예시]
- 이 버튼을 더 크게 만들고 화면 아래에 고정해줘.
- 정답 보기 눌렀을 때 일본어 뜻과 예문이 카드 아래에 펼쳐지게 바꿔줘.
- 나만의 단어장 화면에 import/export 안내 문구를 더 명확하게 넣어줘.

[마지막 핵심]
Cursor는 완전 자동 개발기가 아니라,
“네가 기준을 주고 단계별로 잘 시키면 빠르게 만들어주는 도구”다.
