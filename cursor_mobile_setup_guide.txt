# ✅ Cursor + Expo + 폰 연결 완전 초보 가이드

이 파일은 개발 처음 하는 사람이
👉 "내 PC에서 만든 앱을 폰으로 바로 확인하는 방법"을
순서대로 그대로 따라할 수 있게 만든 단계별 가이드입니다.

--------------------------------------------

# ✅ 전체 흐름

1. PC 준비
2. 프로젝트 실행
3. 폰 앱 설치
4. 같은 네트워크 연결
5. QR 코드로 앱 실행

--------------------------------------------

# ✅ STEP 1. PC 준비

## 1️⃣ Node.js 설치

👉 먼저 Node.js 설치 필요

https://nodejs.org

설치 후 터미널에서 확인:

node -v
npm -v

정상적으로 나오면 OK

--------------------------------------------

## 2️⃣ 프로젝트 폴더 열기

Cursor 또는 VSCode에서

👉 프로젝트 폴더 열기

예:

/topik-repetition-app

--------------------------------------------

# ✅ STEP 2. 프로젝트 실행

터미널 열기 (VSCode 안에서 가능)

## 1️⃣ 패키지 설치

npm install

## 2️⃣ Expo 서버 실행

npx expo start


👉 실행하면 이런 거 뜸

- QR 코드
- localhost 주소

이 상태 유지해야 함 (끄면 앱 안됨)

--------------------------------------------

# ✅ STEP 3. 폰 준비

## 1️⃣ Expo Go 설치

📱 아이폰 → App Store
📱 안드 → Play Store

👉 "Expo Go" 검색 후 설치


--------------------------------------------

# ✅ STEP 4. 같은 Wi-Fi 연결

👉 중요

✔ PC
✔ 폰

둘 다 같은 와이파이 연결해야 함


예:

PC → 집 와이파이
폰 → 집 와이파이


❌ 다른 네트워크면 안됨

--------------------------------------------

# ✅ STEP 5. 앱 실행 (핵심🔥)

방법 1 (추천)

👉 Expo Go 실행
→ "Scan QR Code" 클릭
→ PC 화면 QR 스캔


👉 그러면

⚡ 앱이 폰에서 바로 실행됨

--------------------------------------------

# ✅ STEP 6. 실시간 반영 (핵심✨)

이제부터는:

1. Cursor에서 코드 수정
2. 저장 (Ctrl + S)

→ 폰 화면 자동 업데이트

👉 이걸 Fast Refresh라고 함


예:

- 버튼 위치 변경
- 색상 변경
- 텍스트 변경

👉 바로 반영됨

--------------------------------------------

# ✅ STEP 7. 안될 때 해결법

## 문제 1. 화면이 안 바뀜

👉 해결

터미널에서:

r

또는

앱 흔들기 → Reload 클릭


--------------------------------------------

## 문제 2. 아예 연결 안됨

👉 확인

✔ 같은 WiFi인가?
✔ PC 방화벽 막힘?


--------------------------------------------

## 문제 3. 캐시 문제

👉 해결

npx expo start -c

--------------------------------------------

# ✅ STEP 8. 가장 중요한 개념

이건 “배포”가 아니다

👉 그냥 PC에서 실행된 앱을

👉 폰이 실시간으로 받아서 보여주는 것


그래서:

✔ 스토어 등록 필요 없음
✔ 인증 필요 없음
✔ 바로 테스트 가능

--------------------------------------------

# ✅ 최종 정리

👉 순서 요약

1. npm install
2. npx expo start
3. Expo Go 설치
4. QR 찍기
5. 폰에서 실행
6. 코드 수정 → 자동 반영


--------------------------------------------

# ✅ 한줄 핵심

👉 배포 없이도
"PC에서 만든 앱을 폰에서 실시간으로 바로 확인 가능하다"
