모바일 / 폰 실행 가이드 (초상세판)

목표:
PC에서 실행한 앱을 스마트폰으로 바로 확인하고,
수정하면 즉시 반영되는 개발 환경을 만드는 것.

--------------------------------------------------
1. 준비물
--------------------------------------------------
- PC (노드 설치 완료 상태)
- Cursor
- 스마트폰
- 같은 Wi-Fi 환경

--------------------------------------------------
2. 프로젝트 생성
--------------------------------------------------
1) 터미널 실행

2) 아래 명령어 실행
npx create-expo-app topik-app

3) 템플릿 선택
→ blank (TypeScript)

4) 프로젝트 폴더 이동
cd topik-app

5) 패키지 설치
npm install

--------------------------------------------------
3. 앱 실행
--------------------------------------------------
아래 명령어 입력

npx expo start

→ QR 코드 화면이 터미널 또는 브라우저에 표시됨

--------------------------------------------------
4. 폰 세팅
--------------------------------------------------
1) Expo Go 앱 설치
- iPhone → App Store
- Android → Google Play

2) 꼭 확인:
PC와 스마트폰이 같은 Wi-Fi에 연결되어 있어야 함

--------------------------------------------------
5. 폰에서 실행
--------------------------------------------------
방법 1 (추천):
- Expo Go 실행
- QR 코드 스캔

방법 2:
터미널에서
- a → Android 실행
- i → iOS 실행

--------------------------------------------------
6. 실행 성공 확인
--------------------------------------------------
폰에 Expo 기본 화면이 나오면 성공

--------------------------------------------------
7. 개발 방식 (핵심)
--------------------------------------------------
코드 수정 → 저장 → 폰 자동 반영

이 흐름 반복

--------------------------------------------------
8. 문제 해결
--------------------------------------------------
문제 1: QR 안됨
→ 같은 Wi-Fi 확인

문제 2: 로딩 안됨
→ npx expo start -c

문제 3: 연결 안됨
→ Expo에서 LAN → Tunnel 변경

--------------------------------------------------
9. 테스트 시 체크
--------------------------------------------------
- 버튼 누르기 편한지
- UI 복잡하지 않은지
- Step 흐름 자연스러운지
- 손가락 터치 UX 괜찮은지

--------------------------------------------------
10. 핵심 요약
--------------------------------------------------
npx expo start → QR 스캔 → 폰에서 실행

코드 수정하면 바로 반영됨

--------------------------------------------------
11. 한 줄 핵심
--------------------------------------------------
Expo는 서버 없이 바로 폰에서 테스트 가능한 환경이다.
